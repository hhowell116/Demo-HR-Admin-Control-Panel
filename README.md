# HR Admin Control Panel (Demo)

> **This repo is for demo purposes only.** Firebase authentication and Firestore are replaced with static demo data. No real employee information is included.

**View the demo site:** [hhowell116.github.io/Demo-HR-Admin-Control-Panel](https://hhowell116.github.io/Demo-HR-Admin-Control-Panel/)

**View the real project code:** [github.com/hhowell116/RCO-HR-Admin](https://github.com/hhowell116/RCO-HR-Admin)

---

A multi-package admin dashboard for the Rowe Casa Organics HR team to manage employee recognition programs and internal TV displays. Built as a monorepo with React 18, TypeScript, Tailwind CSS, Firebase Authentication, Cloud Firestore, and Firebase Hosting (multi-site). Features role-based access control (IT Admin, HR, C-Suite), real-time Firestore syncing between admin edits and Airtame-connected display screens, campaign management for birthdays/anniversaries/rockstars, employee photo submissions, and configurable display settings.
